## Issue #1: Hero Slide 1 — Brand Identity Missing

### Current Copy
```
"ทำให้การดูแลบ้านเป็นเรื่องง่าย"
```

### Problem Analysis
- ❌ Generic — could be ANY cleaning brand
- ❌ Doesn't mention Captain Maid
- ❌ No emotional hook or brand personality
- ❌ Passive voice ("ทำให้") — weak brand presence

### Suggested Fix
```
"Captain Maid ทำให้การดูแลบ้านสวยสะอาด เป็นเรื่องง่าย"
```

**Or (More Emotional):**
```
"Captain Maid — ทำความสะอาดบ้านสวยสะอาด ไม่เหนื่อย ไม่เครียด"
```

### Why This Works
- ✅ Leads with brand name (Captain Maid)
- ✅ Adds emotional benefit ("ไม่เหนื่อย ไม่เครียด")
- ✅ Natural Thai rhythm (วรรค — parallel structure)
- ✅ Clear value proposition in 1 line

### Tone Note
Tagline should sound approachable, trustworthy, not corporate. This version feels like a helpful neighbor recommending a product she uses.

## Issue #2: Hero Slide 2 — Product Range (Missing Brand Reference)

### Current Copy
```
"โซลูชันทำความสะอาด
ครบทุกมุมของบ้าน"
```

### Problem Analysis
- ❌ "โซลูชัน" is technical jargon, not conversational Thai
- ❌ Doesn't mention Captain Maid or brand identity
- ❌ Reads like B2B marketing, not B2C

### Suggested Fix
```
"สินค้าทำความสะอาด Captain Maid
ครบครันทั้งห้องนอน ห้องครัว ห้องน้ำ"
```

**Or (Premium Positioning):**
```
"Captain Maid มีทั้งหมด
สำหรับทุกมุมของบ้านคุณ"
```

### Why This Works
- ✅ Opens with Captain Maid brand
- ✅ "ครบครัน" (complete/comprehensive) sounds more natural than "โซลูชัน"
- ✅ Specific rooms feel more relatable than "มุม" (corners)
- ✅ "บ้านคุณ" (your home) creates personal connection

### Tone Note
Thai customers prefer specific examples over abstract words. Listing actual rooms (bedroom, kitchen, bathroom) resonates more than "all areas."

## Issue #3: Hero Slide 3 — Safety (Missing Captain Maid Identity)

### Current Copy
```
"ปลอดภัยสำหรับทุกคน
ในครอบครัว สบายใจได้ทุกวัน"
```

### Problem Analysis
- ❌ Generic safety message (all brands say this)
- ❌ Doesn't connect to Captain Maid's premium positioning
- ❌ Missing brand credentials/proof

### Suggested Fix
```
"Captain Maid ปลอดภัยสำหรับเด็ก สัตว์เลี้ยง
และแม่บ้านทุกคนในครอบครัว"
```

**Or (With Trust Signal):**
```
"ผลิตจากส่วนผสมธรรมชาติ
ปลอดภัยสำหรับเด็ก สัตว์เลี้ยง และทุกคนในบ้าน"
```

### Why This Works
- ✅ Opens with Captain Maid brand name
- ✅ Mentions specific people (kids, pets) — shows thoughtfulness
- ✅ Adds ingredient credibility ("ส่วนผสมธรรมชาติ")
- ✅ Feels more like a product benefit, not generic claim

### Tone Note
Thai families are very concerned about safety for children and pets. Being specific about who it's safe for (not just saying "safe") builds trust. Adding "natural ingredients" reinforces premium positioning.

## Issue #4: Hero Slide 4 — Technology (Unclear Brand Connection)

### Current Copy
```
"เทคโนโลยีทำความสะอาดขั้นสูง
ผสานพลังจากธรรมชาติ"
```

### Problem Analysis
- ❌ "เทคโนโลยี" is abstract — doesn't explain what Captain Maid does
- ❌ "ผสาน" (merge) is awkward — sounds like translation, not natural Thai
- ❌ No mention of Captain Maid or brand benefit

### Suggested Fix
```
"Captain Maid รวมเทคโนโลยีการทำความสะอาด
กับพลังจากธรรมชาติ"
```

**Or (Stronger Positioning):**
```
"ทำความสะอาดลึก ทำให้ยังวิ่วสมส่วน
ด้วยเทคโนโลยี Captain Maid และธรรมชาติ"
```

### Why This Works
- ✅ Opens with Captain Maid brand
- ✅ "รวม" (combine) sounds more natural than "ผสาน"
- ✅ Emphasizes benefit (deep clean + nature)
- ✅ Creates modern + eco-friendly positioning

### Tone Note
Thai consumers like understanding the "why" behind technology. Connecting it to natural ingredients makes tech feel less intimidating and more premium.

## Issue #5: Hero Slide 5 — Trust/Brand Story (Weak Brand Signal)

### Current Copy
```
"คุณภาพที่คุณไว้วางใจได้
ในทุกการดูแลบ้าน"
```

### Problem Analysis
- ❌ Generic trust message (all brands claim this)
- ❌ Doesn't mention Captain Maid name or heritage
- ❌ Missing credentials or proof points
- ❌ No emotional resonance with brand story

### Suggested Fix
```
"Captain Maid — แบรนด์ที่ไว้วางใจได้
ของแม่บ้านไทยทั่วประเทศ"
```

**Or (With Heritage):**
```
"Captain Maid ทำความสะอาดให้สุขสวัสดิ์
เพราะเราใจใส่บ้านของคุณ"
```

### Why This Works
- ✅ Opens with Captain Maid brand + explicit trust signal
- ✅ "แม่บ้านไทยทั่วประเทศ" (Thai mothers nationwide) creates community feeling
- ✅ Shows social proof without needing reviews
- ✅ "ใจใส่" (genuine care) is emotionally resonant in Thai culture

### Tone Note
Thai customers value family, community, and trust earned through generations. Referencing "Thai mothers" and "genuine care" connects emotionally more than generic "quality" statements.

## Issue #6: Products Section — CRITICAL: Internal Copy Must Be Removed

### Current Copy (VISIBLE TO USERS ❌)
```
"ภาพแพ็กเกจทั้งหมดมาจากโฟลเดอร์ assets ที่ผู้ใช้กำหนด 
และใช้ตามต้นฉบับโดยไม่แทนด้วยสต็อก"
```

### Problem Analysis
- 🔴 **CRITICAL:** This is internal development/design documentation
- ❌ Users don't care about file folders or asset management
- ❌ Confusing technical jargon ("assets", "ต้นฉบับ")
- ❌ Unprofessional — looks like a mistake/bug

### Required Fix
**Remove entirely or replace with:**

```
"สินค้าทำความสะอาด Captain Maid
ที่ช่วยให้บ้านคุณสะอาด สวย และปลอดภัย"
```

**Or (Product-Focused):**
```
"ค้นพบสินค้าทำความสะอาดพรีเมียม
ที่เลือกโดยครอบครัวไทยอย่างมีสติ"
```

### Why This Works
- ✅ Removes confusing internal copy
- ✅ Replaces with product benefit (clean, beautiful, safe)
- ✅ Natural, customer-focused Thai messaging
- ✅ Maintains brand positioning

### Tone Note
This is a **ship-blocking issue**. Internal copy MUST NOT be visible to end users. Replace with proper product copy immediately.

## Issue #7: CTA Section — CRITICAL: Non-Standard Thai Word "พียม"

### Current Copy
```
"ยกระดับการทำความสะอาดให้เป็นเรื่องประจำวันแบบพียม"
```

### Problem Analysis
- 🔴 **CRITICAL:** "พียม" is NOT standard Thai language
- ❌ Thai users will not understand this word
- ❌ Looks like a typo or corrupted text
- ❌ Destroys credibility and brand trust
- ❌ May be a romanization error or internal jargon

### Required Fix
```
"ทำให้การทำความสะอาดเป็นประจำวัน
อย่างเรียบง่ายและมีความสุข"
```

**Or (Stronger Emotional Appeal):**
```
"ยกระดับการทำความสะอาดบ้าน
ให้เป็นเรื่องง่าย สบาย สุขสวัสดิ์"
```

**Or (Luxury Positioning):**
```
"บ้านสะอาดและสวยงาม
เป็นแบบคุณทุกวัน"
```

### Why This Works
- ✅ Removes incomprehensible word
- ✅ Uses natural Thai adverbs ("อย่างเรียบง่าย" = simple/easy)
- ✅ Adds emotional benefit (happiness, comfort, beautiful)
- ✅ Maintains CTA momentum ("ยกระดับ" = elevate)

### Tone Note
This is a **critical error**. Non-Thai words destroy brand credibility instantly. Thai consumers are very sensitive to language quality. Fix immediately before launch.

## Issue #8: CTA Section — CRITICAL: Internal Design Copy

### Current Copy
```
"หน้าแลนดิ้งที่ออกแบบมาให้ขายแบรนด์อย่างชัดเจน 
สวย และลดแรงเสียดทานในการตัดสินใจ"
```

### Problem Analysis
- 🔴 **CRITICAL:** This is internal design rationale, NOT customer copy
- ❌ "ลดแรงเสียดทาน" (reduce friction) is UX jargon, not customer language
- ❌ "ออกแบบมา" (designed to) is meta-commentary on the page itself
- ❌ Breaks fourth wall — users shouldn't be told how page was "designed to sell"
- ❌ Sounds manipulative and unprofessional

### Required Fix
```
"ค้นพบ Captain Maid วันนี้
และรับความหมุ่นใจจากผลิตภัณฑ์ที่คุณเชื่อถือได้"
```

**Or (Benefit-Driven CTA):**
```
"เริ่มต้นการทำความสะอาดแบบ Captain Maid
บ้านสะอาด ครอบครัวสุขสวัสดิ์"
```

**Or (Action-Oriented):**
```
"ดูสินค้าทั้งหมด หรือติดต่อเรา
เพื่อคำแนะนำจากผู้ใช้ Captain Maid"
```

### Why This Works
- ✅ Removes internal design commentary
- ✅ Focuses on customer benefit, not design philosophy
- ✅ Includes clear CTA (shop, contact)
- ✅ Builds trust through social proof ("ผู้ใช้")

### Tone Note
**Ship-blocking issue.** Never expose internal design/UX rationale to customers. It breaks the brand experience and sounds amateurish. Replace with customer-focused value proposition.
